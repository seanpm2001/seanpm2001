# Old versions of README

This directory houses old versions of my GitHub profile README.md file in over 109 different languages. Each language has its own separate folder, and the original English 1-100 directory is present, along with a notes folder regarding translations.

## Complete translations

Currently, English is the only complete translation, and it has few errors. There are many errors with translations in other languages, and fluent native speakers of other languages are encouraged to submit reports of translation errors and syntax bugs.

## Notes

Note: due to errors with machine translation, markdown formatting is screwed up, and links, dividers, headings, and other formatting are damaged or missing in all non-English translations. These are tedious to fix, and I don't even know where to begin on how to fix these errors on right-to-left languages like Arabic.
